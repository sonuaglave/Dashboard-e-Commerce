# Ecommerce Dashboard (Flask + HTML + CSS + JavaScript)

A simple and modern **Ecommerce Admin Dashboard** built using Flask for backend and HTML, CSS, JavaScript for frontend.

---

## 🚀 Project Overview

This project is a beginner-friendly ecommerce admin dashboard that allows you to manage and visualize key store data like products, orders, customers, and sales.

It is built using:

- Python (Flask)
- HTML5
- CSS3
- JavaScript

---

## ⚙️ Installation & Setup

### 1. Clone or download the project

```bash
cd ecommerce-dashboard
```

### 2. Install Flask

```bash
pip install flask
```

### 3. Run the application

```bash
python app.py
```

---

## 🌐 Run in Browser

After running the server, open:

```
http://127.0.0.1:5000
```

---

## 🧠 How It Works

- Flask runs the backend server
- `app.py` connects to HTML using `render_template()`
- HTML files are stored in `/templates`
- CSS & JS are stored in `/static`
